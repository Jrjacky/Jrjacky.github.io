链接也是一种文本，所以它也可以设置所有文本可以设置的属性
但是，链接有四种状态：
  1. 未访问过（link）
  2. 已访问过（visited）
  3. 鼠标在链接上（hover）
  4. 鼠标点击链接（active）
这些状态在css设置时，用冒号与a链接在一起，例如：a:link
当然，如果你想使用class选择器的话，就在冒号前面写上class选择器的名字，例如：a.lianjie:visited
我们来实现一个链接，这个链接未访问时是黑的，访问后是绿的，鼠标移动到上面时是粉色的，点击时是蓝色的
怎么操作呢？
很简单，只需要在css中写上：
a.test:link{
    color:#000000;
}
a.test:visited{
    color:#00FF00;
}
a.test:hover{
    color:#FF00FF;
}
a.test:active{
    color:#0000FF;
}
 在html中写上： < a class="test" href="html&css教程.html">这是一个链接< /a>
 结果为： 这是一个链接
当然，除了颜色之外，你还可以改变字体、文字大小、大小写、粗细等等一系列的文字属性
