基本语法

css的基本语法包含两部分：选择器和声明，声明的部分又包含属性和值 基本的格式是这样的：
选择器{
    属性:值;/*这就是一个声明*/     属性:值;......
}
 选择器其实就代表着html标签（就是尖角括号中的p、div、span等等）
如果想同时设置好几个html标签，也可以用逗号隔开
举个例子：
p{
    color:red;/*这句话代表把字体颜色设置为红色*/
    background-color:green;/*这句话代表把背景颜色设为绿色/
}
 然后我们在html代码中写入（或者其他你想写的东西，这只是一个示范）：
< p>红配绿，臭狗屁< /p>
