css的创建一共有3中类型，我们分别来讲述

内嵌式

这是css最基本的创建方法，就是直接写在html代码中！
这种类型我自己比较不推荐，因为这样很难查错其实是因为我的强迫症
使用方法就是直接在html中写上： < xxx style="..."> ... < /xxx>
在style后面写上你的css代码就可以啦！

内部样式表

内嵌式太麻烦了，万一同一个样式我要使用好几遍，咋办？
我们可以使用内部样式表！
我们直接在html代码中写上：
< head>
< style>
...
< /style>
< /head>
在 < style> 和 < /styles> 之间写上你的css代码就可以了

外部样式表

如果一种样式需要用于好几个不同的页面，咋办？
很简单，直接使用外部式样式表！
使用方法很简单：
  1. 创建一个css文件，写入css代码
  2. 在html的内部写上：
    < head>
    < link rel="stylesheet" type="text/css" href="xxx.css">
    < /head>
    其中，xxx.css就是你的css文件
